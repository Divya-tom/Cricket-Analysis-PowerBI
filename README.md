📊 Cricket Analysis – Data Ingestion Using URL
🔹 Project Overview

This project demonstrates data ingestion from a web URL into Power BI using HTML extraction. The dataset was cleaned using Power Query and analyzed using DAX measures.

🔹 Tools Used

Power BI

Power Query (M Language)

DAX

Web Connector

🔹 Key Features

Total Matches KPI

Win Percentage Analysis

Run Rate vs Runs Scatter Analysis

Performance by Opposition

Interactive Slicers

🔹 Outcome

An interactive dashboard providing insights into T20 International performance trends.
